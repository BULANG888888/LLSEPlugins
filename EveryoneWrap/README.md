# EveryoneWrap

玩家公共传送点  
感谢腐竹 **Z.**（2775497761） 给出的开发建议~

## 介绍

### Features

- 所有玩家都可以添加 Warp
- Warp 无法传送，纯生存服务器首选
- 独特的导航功能
- 玩家只能删除他们自己创建的 wrap，OP 可以任意删除
- Warp 没有防重名机制
